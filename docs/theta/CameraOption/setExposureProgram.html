  1. dynamic programValue

exposure program is documented here Must be one of 1, 2, 3, 4, 9.

  • 1 : Manual program - Manually set the ISO sensitivity (iso) setting, shutter speed (shutterSpeed) and aperture (aperture, RICOH THETA Z1 or later).
  • 2 : Normal program - Exposure settings are all set automatically.
  • 3 : Aperture priority program - Manually set the aperture (aperture). RICOH THETA Z1 or later.
  • 4 : Shutter priority program - Manually set the shutter speed (shutterSpeed).
  • 5 : ISO priority program - Manually set the ISO sensitivity (iso) setting.
